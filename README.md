# JavaScript Window Methods Practice
This is a simple front-end project to practice and demonstrate the use of JavaScript `window` object methods.

## Features
- Show alert messages
- Get user confirmation
- Prompt user input
- Display browser window size
- Open external links
- Reload page
- Scroll functionality

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)

## Project Structure
- index.html → Main HTML file
- script.js → JavaScript logic
- css/ → Styles 

## Purpose
This project is created as a learning exercise to understand how browser APIs work, especially the `window` object.
